# SoccerStat

Тестовое задание для стажировки Frontend (SimbirSoft)

## Описание
Веб-приложение для просмотра футбольной статистики: лиги, команды, календари матчей.

### Основной функционал
- Просмотр списка лиг и команд
- Поиск по названию
- Календарь лиги и команды с фильтрами по датам
- Отображение статуса матча, счёта, времени (локальное)
- Адаптивный дизайн (мобильные, планшет, desktop)
- Обработка лимита API (10 запросов/минуту)

### Технологии
- React + Vite
- Tailwind CSS
- Axios + Proxy
- React Router
- LocalStorage (кэширование)

## Как запустить

```bash
npm install
npm run dev
